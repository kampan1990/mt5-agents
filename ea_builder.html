📖 QUICK GUIDE

1. เลือก Strategy

เลือกกลยุทธ์ที่ต้องการ จากนั้นปรับ parameters ของ indicator

2. ตั้งค่า Lot & Risk

Fixed Lot = lot คงที่
% Balance = คำนวณจาก balance
Risk % = คำนวณจาก SL

3. ตั้งค่า SL/TP

SL และ TP เป็น points
1 pip XAUUSD = 100 points
1 pip EURUSD = 10 points

4. Generate → Copy

กด Generate Code แล้ว Copy MQL5 นำไป paste ใน MetaEditor แล้ว Compile
